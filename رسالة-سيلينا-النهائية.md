### C2 ❌ Authentication System مكسور بالكامل

**الملف:** `packages/api/main.py` + `packages/api/auth.py`
**الخطورة:** أي طلب `/auth` يسبب `TypeError` crash

**المشكلة:**
```python
# السطر المكسور — لم يتغير من v5
fastapi_users = FastAPIUsers(
    get_user_manager=lambda: None,  # ← CRASH
    auth_backends=[auth_backend],
)
# لا يوجد User model، لا SQLAlchemy table
```

**الحل:**
```python
from fastapi_users.db import SQLAlchemyUserDatabase

class User(SQLAlchemyBaseUserTableUUID, Base):
    pass

async def get_user_db(session=Depends(get_async_session)):
    yield SQLAlchemyUserDatabase(session, User)

async def get_user_manager(user_db=Depends(get_user_db)):
    yield UserManager(user_db)

fastapi_users = FastAPIUsers(get_user_manager, [auth_backend])
```

### C3 ❌ CORS Configuration مكسور — `allow_origins=['*']` مع `credentials=True`

**الملف:** `packages/api/main.py`
**الخطورة:** المتصفح يرفض كل الطلبات — حتى الـ `/v1/chat/completions` الجديد

**المشكلة:**
```python
CORSMiddleware(
    allow_origins=["*"],      # ← ممنوع مع credentials=True
    allow_credentials=True,   # ← المتصفحات ترفض
)
```

**الحل:**
```python
CORSMiddleware(
    allow_origins=[
        "http://localhost:3000",
        "https://yourdomain.com",
    ],
    allow_credentials=True,
    allow_methods=["GET", "POST", "DELETE"],
    allow_headers=["Authorization", "Content-Type"],
)
```

### C4 ❌ Race Condition — `os.environ` Mutation داخل Async

**الملف:** `packages/core/orchestrator.py`
**الخطورة:** طلبان متزامنان = يفسد كل منهما الآخر

**المشكلة:**
```python
# يعدّل Environment Variables العالمية — خطير جداً
os.environ["OPENAI_BASE_URL"] = selected_tier["base_url"]
os.environ["OPENAI_API_KEY"]  = selected_tier["api_key"]
os.environ["DEFAULT_MODEL"]   = selected_tier["models"][0]
```

**الحل:**
```python
# مرر الـ Config كـ Parameter مباشرة:
llm_config = {
    "base_url": tier["base_url"],
    "api_key":  tier["api_key"],
    "model":    tier["models"][0],
}
await agent.execute(task, memory, llm_config=llm_config)
```

### C5 ❌ SHA256 كـ Embedding — Semantic Search معطل

**الملف:** `packages/core/memory/skill_memory.py`
**الخطورة:** Skill Memory لا يعمل كـ Vector Search — "coding" و "programming" Cosine = Random

**المشكلة:**
```python
# هذا HASH وليس Embedding حقيقي
def _simple_embed(self, text):
    hash_bytes = hashlib.sha256(text.encode()).digest()
    # لا توجد علاقة دلالية بين الـ Vectors
```

**الحل:**
```python
# استخدم FastEmbed (مجاني، بدون GPU)
from fastembed import TextEmbedding

model = TextEmbedding("BAAI/bge-small-en-v1.5")  # 384-dim

def embed(self, text: str) -> list[float]:
    return list(model.embed([text]))[0].tolist()
```
